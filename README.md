## 기능 목록

- [ ] 중복되지 않는 6개의 로또 번호를 뽑는 기능
- [ ] 당첨번호를 입력하는 기능
- [ ] 보너스 번호를 입력하는 기능
- [x] 당첨번호가 6개인지 검사하는 기능
- [x] 당첨번호가 중복되지 않는 지 검사하는 기능
- [x] 당첨번호가 1부터 45 사이의 숫자인지 검사하는 기능
- [ ] 사용자가 구매한 로또 번호와 당첨 번호를 비교하는 기능
- [ ] 로또를 구매하는 기능
- [ ] 로또 당첨내역과 수익률을 출력하는 기능
